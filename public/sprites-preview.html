Code Quest — Character Sprites

All 12 theme characters. Each SVG is authored pointing east (→) on a 48×48 grid. Rotation is pure CSS transform: rotate() — same technique used in canvas via ctx.rotate().

Canvas integration: an <img> is loaded from the SVG data URL once, then drawn with ctx.save() → ctx.translate(cx, cy) → ctx.rotate(θ) → ctx.drawImage(img, -r, -r, size, size) → ctx.restore(). No emoji platform quirks — every character reads correctly in every direction on every device.
